# Crystal Alien 👾

Crystal Alien is a small 2D platformer game developed in Python using **Pygame Zero**.

## 🎮 Description

Crystal Alien is a functional 2D platformer where players control an alien character to navigate through levels, collect crystals, and avoid enemies. This project was developed to demonstrate core software engineering and game development concepts.

## 🛠 Technologies

* Python
* Pygame Zero

## 📦 Requirements

* Python **3.10 – 3.12 recommended**
  (Python 3.13+ may cause compatibility issues with pygame)

You can check your Python version with:

```
python --version
```

## ⬇️ Download the project

Clone the repository from GitHub:

```
git clone https://github.com/brasarlington/crystal-alien.git
cd crystal-alien
```

## ⚙️ Create a virtual environment

Using a virtual environment prevents dependency conflicts.

### Linux / macOS

```
python -m venv venv
source venv/bin/activate
```

### Windows

```
python -m venv venv
venv\Scripts\activate
```

## 📥 Install dependencies

```
pip install pgzero
```

## ▶️ Run the game

```
pgzrun game.py
```

## 🎮 Controls

* Arrow keys → move the alien
* Mouse → Menu interaction
