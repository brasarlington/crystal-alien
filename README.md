# Crystal Alien 👽💎

*A professional 2D platformer adventure built with Python and Pygame Zero.*

## 📖 Project Overview
Crystal Alien is a functional 2D platformer where players control an astronaut character to navigate through levels, collect crystals, and avoid enemies like slimes and bats. This project was developed to demonstrate core software engineering and game development concepts, including:

* **Object-Oriented Programming (OOP):** Modular design for game entities such as the Player, Enemies, and Collectibles.
* **Physics & Kinematics:** Custom implementation of gravity, jump velocity, and acceleration.
* **Collision Systems:** Robust AABB collision detection for platforms and hazards.
* **Asset Pipeline:** Organized management of sprites, sound effects, and background music.

---

## 📂 Project Structure
A look into how the project is organized:
* `game.py`: Main entry point containing the game loop and core logic.
* `images/`: High-quality 2D sprites and backgrounds.
* `music/`: Atmospheric background tracks.
* `sounds/`: Trigger-based sound effects (jump, collect, hit).
* `organize_sprites.py`: Utility script for asset management.

---

## 🎮 Controls
* **Arrow Keys (Left/Right):** Movement.
* **Up Arrow:** Jump.
* **Mouse:** Menu navigation and interaction.
