# Crystal Alien


## 📖 Project Overview
Crystal Alien is a functional 2D platformer where players control an alien character to navigate through levels, collect crystals, and avoid enemies. This project was developed to demonstrate core software engineering and game development concepts.

---

## 📂 Project Structure
* `game.py`: Main entry point containing the game loop and core logic.
* `images/`: High-quality 2D sprites and backgrounds.
* `music/`: Atmospheric background tracks.
* `sounds/`: Trigger-based sound effects.

---

## 🎮 Controls
* **Arrow Keys:** Movement and Jumping.
* **Mouse:** Menu interaction.

---

## 🚀 Setup & Execution

The game requires the pgzero library to handle the game loop and rendering.

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/brasarlington/crystal-alien.git](https://github.com/brasarlington/crystal-alien.git)
   cd crystal-alien
2. **Install the pgzero module:** 
   ```bash
   pip install pgzero
3. Launch the game using the Pygame Zero runner:
   ```bash
   pgzrun game.py
